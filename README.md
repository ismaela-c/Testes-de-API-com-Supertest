Testes de API com Supertest – Swagger Petstore

Projeto de automação de testes de API utilizando Supertest e Jest, realizando operações de CRUD na entidade User da API pública Swagger Petstore.

**Tecnologias utilizadas**
- ✅ Node.js
- ✅ Jest
- ✅ Supertest
- ✅ API pública: https://-petstore.swagger.io

**Objetivo**

Validar o funcionamento dos endpoints da entidade User, cobrindo:
✅ Criação de usuário (POST)
✅ Consulta de usuário (GET)
✅ Atualização de usuário (PUT)
✅ Remoção de usuário (DELETE)

Os testes foram implementados também utilizando abordagem Data Driven, com massa de dados externa.
